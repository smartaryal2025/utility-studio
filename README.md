# Utility Studio

A privacy-first, 100% offline suite of web-based productivity tools. 

Utility Studio brings heavy-duty processing directly to the local device browser using modern web technologies like WebAssembly (WASM) and Service Workers. No server uploads, no hidden tracking, and full offline capability.

## 🛡️ Core Philosophy: Zero-Upload Architecture
Most online converters force users to upload sensitive documents to remote servers, exposing private data and wasting bandwidth. Utility Studio processes all files—including PDFs, images, and videos—entirely client-side. Your data never leaves your device.

## 🧰 Included Tools
* **PDF Toolkit Pro:** Securely merge, split, and edit sensitive PDFs locally.
* **Media Studio:** Extract audio and convert video formats using in-browser FFmpeg.
* **Image Studio:** Crop and resize photos locally without leaking hidden EXIF location data.
* **QR & Barcode Lab:** Generate static, untracked Wi-Fi and vCard codes securely.
* **Unicode Tools:** Convert Preeti to Unicode instantly with zero latency.
* **Date Converter:** Lightning-fast offline conversion between Bikram Sambat (B.S.) and Gregorian (A.D.).
* **Unit Converter:** Calculate global metrics and local Nepali units (Ropani, Bigha, etc.) instantly.

## 🚀 Technical Stack
* **Frontend:** HTML5, CSS3 (CSS Variables for dynamic Dark/Light theming), Vanilla JavaScript.
* **Offline Infrastructure:** Service Workers (PWA capabilities for airplane-mode access).
* **Processing Engines:** WebAssembly (FFmpeg for video), PDF-lib (for documents), HTML5 Canvas (for image manipulation).
* **Hosting:** Served globally via Cloudflare Pages.
